# Anforderungen

- Die App sollte eine Seite haben mit zwei Tabs

## Erster Tab

- Name: Brutto -> Netto
- Formular
  - Input: Bruttobetrag
  - Chooser für Steuersatz (Dropdown oder Radio)
    - 7%
    - 19%
  - Textausgabefeld: Steuerbetrag
  - Textausgabefeld: Nettobetrag

## Zweiter Tab

- Name: Netto -> Brutto
- Formular
  - Input: Nettobetrag
  - Chooser für Steuersatz (Dropdown oder Radio)
    - 7%
    - 19%
  - Textausgabefeld: Steuerbetrag
  - Textausgabefeld: Bruttobetrag
